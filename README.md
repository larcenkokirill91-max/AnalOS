Как запустить:
1. Установите QEMU если ещё он не установлен.
2. Введите команду make clean && make && make run предварительно зайдя в папку с .bin файлом, также для интереса вы можете посмотреть код ведь все файлы открыты для доступа.
3. Посмотрите на 3 разноветных квадрата.
